# Ansible
## What's this?
This directory contains all the ansible playbooks, roles and other ansible-related files that were used in this project.  
It also has [the rulebooks](eda_rulebooks) that were used by [Event-driven-ansible](../concepts_explained/event-based-pipelines) to trigger the playbooks.

## Requirements
- Python    (tested with Python 3.10.12)
- pip       (tested with pip 22.0.2)
- ansible   (tested with ansible community 9.4)
